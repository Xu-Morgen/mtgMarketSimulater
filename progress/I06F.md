## 特殊点

- 服务器仍是身份与角色权限的唯一权威；前端仅改善路由体验，管理 API 继续由 Fastify RBAC 复核。
- access token 只保存在浏览器内存；refresh token 为 HttpOnly Cookie，CSRF 值以非 HttpOnly Cookie + Header 校验。
- 自动化验证包括玩家/管理员登录、刷新、退出、错误密码与管理 API 403；人工验收由用户于 2026-07-24 确认通过。
