## 特殊点

- 架构/权威边界：目录 API 仅读取 SQLite 与本地图片缓存元数据；外部 Scryfall 导入仍留在 I09 后台任务，浏览器不会触发外部请求。
- 契约/迁移：迁移号为 `0007_catalog.sql`；`(printing_id, finish)` 唯一，工艺仅允许 `nonfoil`、`foil`、`etched`。
- 数据兼容性：空库启动会自动应用该迁移；已有数据库保留既有数据并新增空目录表。
- 安全：目录读取要求有效 Bearer 会话；人工测试/运营例外必须是 `manual-test` 且 `isManualException = true`，不能作为外部参考价资料。
- 已验证：`pnpm --filter @mtg-market/api test -- catalog-routes.test.ts`、相关三个 workspace 的 TypeScript 检查通过；全局 `pnpm check` 见本次变更的最终验证。
