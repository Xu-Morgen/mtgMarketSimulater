- `apps/web/app/styles.css`、`apps/web/tsconfig.json`：按新增的 Prettier 规则规范化既有前端样式和 TypeScript 配置。

- 权威边界：Web 只读取 `NEXT_PUBLIC_API_BASE_URL`；AI 密钥仅在 `apps/ai` 的服务端配置中解析。
- 数据兼容性与迁移：无数据库迁移或运行时数据变更。
- 幂等与审计：I01 未新增业务写端点，故无幂等或审计语义变更。
- 已验证：`pnpm install --offline`、`pnpm check`、`pnpm test`、`pnpm format:check` 与 `pnpm build` 均通过；Fastify 的 `/health` 返回 `{"status":"ok","storage":"sqlite-wal"}`，Next.js 首页也可正常响应。
